###Technology is used:

+ Frontend: HTML, CSS, ANT Design, React

+ Backend: JavaScript, nodejs, expressJS, MongoDB

+Home Page

+Products Page

+Products Detail Page

+Order Page

+Sign-in/Sign-up

BE:
